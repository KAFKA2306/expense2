# 支出データ分析

## 💡 このツールでできること

- 家計の傾向が分かる：月ごとの支出推移や、1年単位の移動平均（12MA）をグラフで確認できます。
- 無駄な出費を見つける：サブスクリプションなどの固定費や、削減しやすい支出（外食・趣味など）を自動でリストアップします。
- 将来の支出を予測する：これまでのデータから、年間の合計支出がいくらになりそうかを予測します。
- サクサク動く：FastAPIとHTMXを使用しており、動作が軽快です。

## 🛠 主な機能

- ダッシュボード: 直近の取引と家計のサマリーを表示します。
- グラフ分析: カテゴリ別の割合や月次の推移を可視化します。
- 固定費チェック: 毎月発生している支払いを特定します。
- データクレンジング: 「振替」や「収入」を除外し、純粋な支出だけを分析対象にします。

## 🚀 使い方

Python環境（uv）があれば、すぐに始められます。

```bash
# セットアップ
uv sync

# アプリの起動
task dev
```

起動後、ブラウザで `http://localhost:8000` にアクセスしてください。
